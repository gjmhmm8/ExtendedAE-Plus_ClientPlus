# ExtendedAE Plus [ClientPlus]
EAEP, 但是纯客户端! _(当然功能很少)_

### 包含功能以及其与 common 版本的差别
- 快速样板上传
  - 在编写样板前打开样板管理终端以创建/同步服务器样板供应器数据
  - 会将选择的供应器写入本地缓存, 在再次打开样板管理终端时自动安置
  - 因为延迟问题, 安置操作可能失败  
    使用 [Shift + Esc] 退出样板管理终端即可保留样板缓存, 再次打开终端会继续尝试安置
- 快速拉取物品
  - 仅在包含库存的 gui 打开时能运作
- 石英切割刀复制方块名
- [Ctrl + 中键] 样板编码终端快捷重命名
- [Ctrl] 点击合成计划开始界面以标记配方自动完成
- 自动切割名称压印模板
  - 推荐与快速重命名配合使用
  - 可以在背包含有可以制作名称压印模板的材料 _(小心你的高级锭!)_  
    与编有自定义名称的物品的编码样板时  
    自动制作名称压印模板
